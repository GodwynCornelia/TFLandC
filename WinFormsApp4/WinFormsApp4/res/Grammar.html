Описание грамматики

Для распознавания конструкции объявления строковой константы в языке Rust используется регулярная (автоматная) грамматика:

Формальное определение: G[Z] = {Vt, Vn, Z, P}

Алфавит терминальных символов (Vt): Vt = { 'const', ':', '&str', 'space', '=', '"', ';', '_', a...z, A...Z, 0...9, '!', '?', ... }

Алфавит нетерминальных символов (Vn): Vn = { <Z>, <ID>, <Space1>, <Eq>, <Bodystring>, <IDRest>, <IDStart>, <QuoteOpen>, <CloseQuote>, <TypeStart>, <Letter>, <Digit>, <Symbol> }

Правила переходов (P):
  • <Z> → 'const' <Space1>
  • <Space1> → ' ' <IDStart>
  • <IDStart> → '_' <IDRest> | <Letter> <IDRest>
  • <IDRest> → '_' <IDRest> | <Letter> <IDRest> | <Digit> <IDRest> | ':' <TypeStart>
  • <TypeStart> → '&str' <Eq>
  • <Eq> → '=' <QuoteOpen>
  • <QuoteOpen> → '"' <BodyString>
  • <BodyString> → <Letter> <BodyString> | <Digit> <BodyString> | <Symbol> <BodyString> | '"' <CloseQuote>
  • <CloseQuote> → ';' END
  • <Letter> → 'a' | 'b' | ... | 'Z'
  • <Digit> → '0' | '1' | ... | '9'
  • <Symbol> → '!' | '?' | '_' | ...
