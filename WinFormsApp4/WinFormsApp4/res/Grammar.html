Описание грамматики

Для распознавания конструкции объявления строковой константы в языке Rust используется регулярная (автоматная) грамматика:

Формальное определение: G[Z] = {Vt, Vn, Z, P}

Алфавит терминальных символов (Vt): Vt = { const, :, &str, space, =, ", ;, [^"], Б, Ц, _ }
Где Б — буквы, Ц — цифры, [^"] — любой символ кроме кавычки.

Алфавит нетерминальных символов (Vn): Vn = { <Z>, <Space1>, <IDStart>, <IDRest>, <TypeStart>, <Eq>, <QuateOpen>, <BodyString>, <BodyStringRest>, <CloseQuote> }

Правила переходов (P):
  • <Z> → const <Space1>
  • <Space1> → space <IDStart>
  • <IDStart> → _ <IDRest> | Б <IDRest>
  • <IDRest> → _ <IDRest> | Б <IDRest> | Ц <IDRest> | : <TypeStart>
  • <TypeStart> → &str <Eq>
  • <Eq> → = <QuateOpen>
  • <QuateOpen> → " <BodyString>
  • <BodyString> → [^"] <BodyStringRest>
  • <BodyStringRest> → [^"] <BodyStringRest> | " <CloseQuote>
  • <CloseQuote> → ; END
