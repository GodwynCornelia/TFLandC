Тема работы

Разработка текстового редактора с функциями языкового процессора. Объявления строковой константы на языке Rust.

Особенности языка

В языке Rust константы представляют собой значения, которые привязаны к идентификатору и не могут быть изменены. Они объявляются с использованием ключевого слова const.

Основные правила объявления строковых констант в Rust:

  • В отличие от переменных (let), тип константы обязательно должен быть указан явно после двоеточия.
  • Для строковых литералов используется тип &str (ссылка на строковый срез).
  • Значение должно быть известно на этапе компиляции.

Формат объявления:

const NAME: &str = "value";

Примеры верных строк из языка

В соответствии с разработанной автоматной грамматикой, процессор будет считать верными следующие записи:

const GREETING: &str = "Hello, Rust!";
const _PRIVATE_MSG: &str = "top secret";
const VERSION1: &str = "1.0.2";
const empty: &str = "";

Синтаксический анализатор проверяет строгое соблюдение последовательности: наличие пробелов, двоеточия, амперсанда перед типом str и завершающей точки с запятой.
